## 📋 Prerequisites

- PowerPoint 2019+ or PowerPoint 365 (Mac/PC)
- A messy presentation that needs fixing (or download our sample)
- Basic PowerPoint navigation skills
- Internet connection for AI image generation

## Step 1: Build Your Design System (0–12 min)

First, establish visual consistency with fonts and colors that work everywhere.

Open your ugly presentation. Create a new slide and access the master slide editor:

```powerpoint
View tab → Slide Master → Edit Master Slides
```

Set up your typography hierarchy:
- **Headlines:** Montserrat Bold, 36pt (download from Google Fonts)
- **Subheads:** Montserrat Medium, 24pt  
- **Body text:** Open Sans Regular, 18pt
- **Captions:** Open Sans Regular, 14pt

Choose your 3-color system:
- **Primary:** #2C3E50 (dark navy - for headlines)
- **Accent:** #3498DB (blue - for highlights and CTAs)
- **Light:** #ECF0F1 (light gray - for backgrounds and dividers)

Apply these to your master slide layouts. Update the title placeholder to use Montserrat Bold in #2C3E50, and body text to Open Sans in the same color.

## Step 2: Apply the 5 Core Layout Templates (12–25 min)

Replace chaotic slides with these proven structures:

**Layout 1: Hero Slide**
- Large headline (40% of slide height)
- Single supporting image (right 60% of slide)
- One-line subtitle below headline

**Layout 2: Problem/Solution Split**
- Left half: problem statement with dark background
- Right half: your solution with light background  
- Use contrasting colors to show the transition

**Layout 3: Three-Point Framework**
- Horizontal layout with 3 equal columns
- Icon + headline + 2 bullet points per column
- Consistent spacing using PowerPoint's align tools

**Layout 4: Data Story**
- Chart takes up 70% of slide real estate
- One key insight as headline above chart
- Source citation in small text at bottom

**Layout 5: Next Steps**
- Numbered list with large, readable numbers (48pt)
- Each step gets 2-3 lines maximum
- Clear call-to-action at bottom

Take your existing slides and force-fit them into these 5 templates. Delete excess text ruthlessly - if it doesn't fit the template, it probably isn't essential.

## Step 3: Generate and Integrate AI Images (25–40 min)

Replace stock photos and clipart with custom AI visuals that match your content.

Use DALL-E 3 (via ChatGPT Plus) or Midjourney for image generation:

```prompt
Create a professional business illustration showing [your concept], 
clean minimal style, blue and navy color scheme, white background, 
suitable for corporate presentation
```

For each slide needing visuals:

1. **Hero slides:** Generate abstract concepts that represent your main theme
2. **Process slides:** Create simple icons or diagrams showing workflow
3. **Data slides:** Generate metaphorical images (growth = climbing stairs, teamwork = puzzle pieces)

Download images in high resolution (1024x1024 minimum). In PowerPoint:

```powerpoint
Insert → Pictures → This Device → Select your AI image
Right-click image → Format Picture → Transparency 10-20%
```

Position images to complement, not compete with text. Use the alignment guides to ensure consistent spacing.

For slides without obvious visual needs, add subtle geometric shapes or gradients as background elements using your brand colors.

## Step 4: Polish with Animations and Consistency (40–52 min)

Add subtle motion that guides attention without being distracting.

Apply entrance animations sparingly:
- **Headlines:** Fade in (0.5 seconds)
- **Bullet points:** Appear with previous (0.3 seconds delay)
- **Images:** Zoom in slightly (0.4 seconds)

```powerpoint
Select element → Animations tab → Fade → Animation Pane → 
Adjust timing to "With Previous" for supporting elements
```

Run through your consistency checklist:
- All headlines use the same font and size
- Consistent margins (use PowerPoint's ruler - 1 inch on all sides)
- Images aligned to the same grid
- Color usage follows your 3-color rule
- No more than 6 lines of text per slide

Fix alignment issues using:

```powerpoint
Select multiple objects → Format tab → Align → 
Choose "Align Left" or "Distribute Horizontally"
```

## Step 5: Ship It (52–60 min)

Prepare multiple formats for different presentation scenarios.

Export your presentation:

```powerpoint
File → Export → Create PDF/XPS → Options → 
Check "Publish what: Slides" → Optimize for "Print"
```

Create a presenter-friendly version:
- Add slide numbers (Insert → Header & Footer → Slide Number)
- Include notes for complex slides (View → Notes Page)
- Save a backup copy with animations removed for older systems

Test your presentation:
- Run slideshow mode on full screen
- Check readability from 6 feet away
- Verify all animations work smoothly
- Ensure slides advance with spacebar/click

🎉 **You're done!** Your presentation now looks like it came from a design agency, not a default PowerPoint template.
