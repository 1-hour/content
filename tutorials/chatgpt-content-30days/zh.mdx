## 📋 前置条件

- ChatGPT 账户（免费版即可）
- Google 表格或类似电子表格工具
- 确定你的细分领域/主题方向（如生产力、健身、职场建议）
- 对目标受众的基本了解

## 第 1 步：构建钩子公式库 (0–15 分钟)

首先，创建一个包含验证有效钩子模板的文档。这些公式之所以有效，是因为它们能激发好奇心、争议或即时价值感。

复制这 8 个爆款钩子公式：

```
公式 1 - 失误型："我在[主题]上[花钱/花时间]，这是我学到的"
公式 2 - 争议型："为什么[流行观点]实际上是[相反观点]"
公式 3 - 清单型："[数字]个[工具/技巧/秘密]让你[具体收益]"
公式 4 - 故事型："我如何在[时间]内通过[方法][取得成果]"
公式 5 - 警告型："在[负面后果]发生前停止[常见行为]"
公式 6 - 技巧型："[数字]个[平台][技巧/窍门]他们不想让你知道"
公式 7 - 对比型："[选项A] vs [选项B]：哪个真正[产生结果]"
公式 8 - 内幕型："关于[主题/行业]没人告诉你的真相"
```

现在告诉 ChatGPT 你的细分领域。使用这个提示词：

```
我的细分领域是[你的领域 - 如"远程工作新手的生产力"]。
我的受众在[主要痛点 - 如"在家工作时保持专注"]方面有困扰。

为我的领域定制，为以下 8 个公式各生成 3 个钩子示例：
[在此粘贴你的 8 个公式]

让每个钩子控制在 15 字以内，并立即明确价值所在。
```

你是否有 24 个钩子示例（每个公式 3 个），都针对你的细分领域且控制在 15 字以内？
<Answer>是的，每个钩子都清楚表明价值并符合我受众的痛点</Answer>
</Checkpoint>

## 第 2 步：生成 30 天内容流水线 (15–30 分钟)

创建让受众持续参与的内容组合。最优秀的创作者使用这个比例：
- 40% 教育型（教程、技巧、操作指南）
- 30% 励志型（故事、收获、幕后）
- 20% 争议型（热门观点、辟谣）
- 10% 推广型（你的产品/服务）

使用这个 ChatGPT 提示词生成你的创意：

```
为[你的细分领域]创建 30 天内容日历。遵循以下比例：
- 12 个教育型帖子（教程、分步指南、工具评测）
- 9 个励志型帖子（成功故事、经验教训、激励）
- 6 个争议型帖子（挑战流行观点、热门观点）
- 3 个推广型帖子（展示专业性、软销售）

为每个帖子提供：
1. 帖子类型
2. 主要话题
3. 解决的目标受众痛点
4. 关键收获

按编号列表格式输出。让话题具体且可操作。
```

审查输出结果。好的话题是具体的（不是"如何提高生产力"而是"如何在 15 分钟内批量回复邮件"）。糟糕的话题是模糊或过于宽泛的。

现在用这个后续提示词完善薄弱话题：

```
第 #[X, Y, Z] 个帖子过于模糊。通过添加以下内容让它们更具体：
- 确切的数字或时间范围
- 具体的工具或方法
- 清晰的前后对比结果

仅重写这 3 个帖子。
```

<Checkpoint>
你的 30 个话题是否都解决受众面临的具体问题，并有他们可以达成的可衡量结果？
<Answer>是的，每个话题都具体可操作，价值明确</Answer>
</Checkpoint>

## 第 3 步：撰写爆款钩子和完整帖子草稿 (30–45 分钟)

将你的话题转化为停止滑动的内容。从 5 个最高价值话题开始（通常是解决紧急问题的教育型内容）。

为每个帖子使用这个提示词：

```
为这个话题写一个完整的社交媒体帖子：[列表中的话题]

按以下结构：
1. 钩子：使用这些公式之一 [从第 1 步粘贴 2-3 个相关钩子公式]
2. 问题：这解决了什么痛点？（1-2 行）
3. 解决方案：主要内容（3-5 个要点或短段落）
4. 行动号召：提问或请求互动

针对以下平台优化：
- 小红书：更多个人叙事，谨慎使用表情符号
- Twitter：简洁有力，适合串推，包含相关标签

写出两个版本。
```

好坏对比如下：

**糟糕钩子：**"今天我想分享一些生产力技巧"
**优秀钩子：**"我在 30 天内试用了 47 个生产力应用，只有这 3 个真正为我节省了时间"

**糟糕内容：**长段落理论
**优秀内容：**有编号的具体行动步骤

对于前 5 个帖子，让 ChatGPT 改进薄弱钩子：

```
为这个钩子的爆款潜力评分（1-10）："[你的钩子]"

如果低于 8/10，使用以下标准重写：
- 立即创造好奇心或争议
- 承诺具体价值（数字、时间范围）
- 直接针对[你的受众]最大困扰
```

## 第 4 步：构建内容日历系统 (45–55 分钟)

创建包含以下列的跟踪表格：
- 日期
- 平台（Twitter/小红书/两者）
- 钩子
- 帖子类型（教育/励志/争议/推广）
- 内容状态（草稿/就绪/已发布）
- 表现备注

战略性地组织发布计划：
- 每周以教育内容开始（周一参与度更高）
- 周中使用争议观点（人们最活跃时）
- 周末用励志内容结束
- 推广帖子每 10 天间隔一次

使用这个 ChatGPT 提示词优化你的计划：

```
我规划了 30 个社交媒体帖子。帮我在 30 天内安排它们以获得最大参与度。

规则：
- 每天发布
- 每周以教育内容开始
- 周二/周三使用争议帖子
- 同一周内不要放 2 个推广帖子
- 整月平衡内容类型

我的帖子类型：[列出你的 30 个帖子及其类型]

给我逐日计划并说明时间选择的理由。
```

<Answer>是的，我有完整的日历，包含战略性的帖子时间安排和多样性</Answer>

## 第 5 步：上线发布 (55–60 分钟)

创建可复用提示词库，这样你就永远不会缺少内容创意。保存这 5 个提示词：

**提示词 1 - 创意生成器：**
```
为[你的细分领域]生成 10 个[内容类型]创意，解决[具体痛点]。每个创意应该在[时间范围]内可操作，并包含具体数字或结果。
```

**提示词 2 - 钩子改进器：**
```
重写这个钩子使其更具爆款潜力："[钩子]"。让它创造好奇心，包含具体数字，并针对[受众]的主要困扰。给我 3 个版本。
```

**提示词 3 - 内容扩展器：**
```
将这个话题"[话题]"转化为完整的社交媒体帖子，包含：爆款钩子、问题陈述、5 个可操作步骤和互动问题。为[平台]撰写。
```

**提示词 4 - 趋势利用器：**
```
[热门话题]现在很流行。创建 5 个内容角度，以有价值的方式将这个趋势与[你的细分领域]连接起来，服务[你的受众]。
```

**提示词 5 - 表现分析器：**
```
我的帖子"[钩子]"获得了[参与数据]。分析它成功/失败的原因，并创建 3 个类似但可能表现更好的帖子。
```

现在用提示词 1 生成下周的内容来测试一个提示词。

🎉 你现在有了 30 天的内容规划、草稿和发布计划，还有一个能按需生成无限创意的系统。

## 🎁 进阶挑战

- **批量创作模式**：在一个 ChatGPT 会话中生成整周的钩子，然后将表现最佳的扩展为完整帖子
- **竞品分析**：向 ChatGPT 输入你领域内的成功帖子，让它识别心理触发因素，然后将这些模式应用到你的内容中
- **季节性内容**：使用你的提示词提前 90 天创建节假日、活动或趋势特定的内容日历
